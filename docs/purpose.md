## Purpose

The primary purpose of v-commerce is to demonstrate:

- Microservices architecture patterns
- Kubernetes deployments and orchestration
- Cloud-native application development
- Observability and monitoring practices
- Tools and technologies commonly used in modern distributed systems

while being accessible and useful to all new Kubernetes users.

### Why does the purpose matter?

We filter and prioritize the work to be done in this repository based on the purpose defined above.
If you wish to make changes to this repository that do not align with the above purpose, we encourage you to maintain your own fork of v-commerce.
